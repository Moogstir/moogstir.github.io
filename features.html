Features That Define

Novelative is for novelists — writers who need control, structure, and visualization far beyond standard word processors.

1. Page-Perfect Engine

What You See Is Exactly What You Print.

Image: Editor Canvas View (Page Break Example)

Canvas Rendering Core

Documents are rendered onto an HTML5 ``, not standard DOM elements. This eliminates layout shifts and guarantees precise, fixed-page formatting, simulating a professional word processor.

Custom Layout Engine

A mathematical engine manually calculates the exact X/Y coordinate for every word, handling line wrapping, justification, and dynamic page breaks instantly as you type.

Visual Ruler System

Fine-tune your typography using a visual ruler. Drag-and-drop margins, first-line indents, and hanging indents—changes immediately trigger a full document recalculation.

2. Deep Interconnectivity

Structure Your World with Smart Logic.

Smart Inheritance

Tags applied to parent folders (e.g., "Draft 1") are automatically inherited by all children files, making project-wide versioning and filtering seamless and automated.

Wiki-Linking & Backlinks

Create internal links by typing `[[File Name]]`. The system automatically tracks backlinks, showing every document that references your current file in the sidebar.

Recursive File Tree

Organize characters, chapters, and lore into a highly nested file tree structure. Supports three node types: Folders, Text Documents, and Whiteboard Canvases.

Image: File Tree & Smart Tagging
3. Visual Thinking

Nonlinear Plotting Meets Physics.

Image: Whiteboard/Graph View in action

The Dynamic Graph View

Visualize your entire project's structure as an interactive network. Switch between **Tag Mode** (clusters) and **Link Mode** (relationship visualization).

Whiteboard Orthogonal Routing

Plot scenes and ideas on an infinite canvas. Connections between cards use smart orthogonal (right-angle) routing for clean, grid-like, and professional flowcharts.

Draggable Entities & Snapping

Drag files directly from the sidebar onto the whiteboard to create linked cards. Cards snap and group together, allowing you to move complex plot segments as one unit.

4. Right Sidebar & Desktop Power

Offline Persistence and Context-Rich Tools.

Global Search & Thesaurus

Project-wide search scans all JSON content. A built-in, offline thesaurus offers instant synonyms without breaking flow.

Contextual Commenting

Highlight text to add comments. The sidebar instantly scrolls the editor to the exact location of the highlighted text for easy review.

Electron & Offline First

Built on Electron and React for a native experience. Data is stored locally in JSON files, requiring zero internet connection to work.

Novelative

A powerful, offline-first writing studio designed for novelists and world-builders.
